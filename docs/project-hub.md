# Fragrance App

Index: [[Projects Index]]

Personal fragrance collection tracker. Mobile-first (Expo / React Native), Supabase backend, shipped in phases. Catalog-first: brand, name, concentration, accords, personal rating. Phase 1 is tagged; Phase 1.5 now has wear logging, Calendar month/by-bottle views, selected-day wear entry, curated accord autocomplete, and local catalog lookup/prefill. Barcode scanning and the shared Supabase catalog remain Phase 2.
